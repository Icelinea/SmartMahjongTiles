## Introduction

Intend to build a mahjong tiles that can control the win rate of the players.

## Develop

1. 接入 ai 模型打牌逻辑
2. 接入人类鼠标互动打牌功能
3. 微调显示界面 - 其他玩家的手牌旋转显示、丢弃区显示、宝牌
4. 重构牌山随机生成功能，改为可训练模型
5. UI 常量文件化存储
